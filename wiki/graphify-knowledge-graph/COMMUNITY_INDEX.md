# SkyHigh Knowledge Graph - Community Index

This document maps the 121 automatically-detected communities to meaningful functional areas.

## Why Communities?

Graphify uses graph clustering to find groups of related code. Communities represent:
- Cohesive features (closely related functions, components, modules)
- Architectural boundaries (where dependencies cluster)
- Refactoring opportunities (tight communities can be extracted or optimized)

The automatic numbering (Community 0, 1, 2...) doesn't reflect purpose. This index maps them to **what they actually do**.
